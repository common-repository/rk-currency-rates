Курс валют национальный банк РК

== Description ==

Информация о курсах обмена валюты, установленных Национальный банк РК (www.nationalbank.kz). Все данные берет с сайта www.nationalbank.kz (XML)

== Installation ==

# Атрибуты:

**font=&quot;14px&quot;** - размер текста
**all=&quot;true&quot;** - показать все валюты
**link=&quot;курсы-валют-рк&quot;** - ссылка на страницу (подробнее)

Установка и использование плагина:

1) Плагины -> Добавить новый -> "Загрузить плагин" или "Установить"
2) Плагины -> Установленные -> Курсы валют РК -> Активировать
3) Страницы
	-> Добавить новую
	-> Заголовок любое (Например: Курсы валют РК)
	-> Текст: [rk-currency-rates font=&quot;14px&quot; all=&quot;true&quot;]
	-> Опубликовать
	-> Копируем ссылку на страницу только что созданный (можно только алиас): курсы-валют-рк
4) Внешний вид
	-> Виджеты
	-> Текст или HTML-код
	-> Заголовок любое (Например: Курсы валют РК)
	-> Текст: [rk-currency-rates font=&quot;12px&quot; link=&quot;курсы-валют-рк&quot;]
	-> Сохранить
